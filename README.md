# 🔗 Social Links Manager (Chrome Extension)

A lightweight Chrome Extension that allows users to store and quickly copy their social media profile links from one place.

---

## 🚀 Features

* Save multiple social media profile links
* One-click copy to clipboard
* Delete and manage saved links
* Simple and user-friendly UI
* Data stored using Chrome Storage API

---

## 🛠 Tech Stack

* HTML
* CSS
* JavaScript
* Chrome Extension (Manifest V3)
* Chrome Storage API

## ▶️ How to Run Locally

1. Download or clone this repository
2. Open Chrome and go to:

   ```
   chrome://extensions/
   ```
3. Enable **Developer Mode** (top right)
4. Click **Load Unpacked**
5. Select the project folder
6. Click on the extension icon and start using it
